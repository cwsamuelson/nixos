## `password-store`
[Password Store Website](http://www.passwordstore.org)
Local storage of passwords and secrets.  Uses Unix philosophy (the local
filesystem) to store, organize, and group passwords.  Additionally uses standard
tooling (i.e. GPG) for encryption.
